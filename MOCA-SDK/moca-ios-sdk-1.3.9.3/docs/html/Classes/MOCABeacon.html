Overview

The MOCABeacon class represents a beacon device. Beacon devices are managed by MOCA Cloud Console and encountered during beacon region monitoring. You do not create instances of this class directly.

MOCABeacon class contains basic device information as registered in the cloud, and when detected it contains CLBeacon object reference with access to device state.
